## Description
This project is an audit of Python as an open-source software. It includes analysis of its origin, license, ecosystem, and comparison with proprietary software. It also includes five shell scripts demonstrating Linux concepts.

## Scripts

### Script 1: System Identity Report
Displays system information such as kernel version, user, uptime, and date.

### Script 2: FOSS Package Inspector
Checks if Python is installed and displays its version.

### Script 3: Disk and Permission Auditor
Shows permissions and disk usage of important system directories.

### Script 4: Log File Analyzer
Counts occurrences of a keyword in a log file.

### Script 5: Open Source Manifesto Generator
Generates a personalized open-source statement based on user input.

## How to Run Scripts

1. Open terminal in Linux (WSL/Ubuntu)
2. Navigate to project folder:
   cd oss-project

3. Give permission:
   chmod +x script1.sh.save

4. Run script:
   ./script1.sh.save

(Repeat for other scripts)

## Requirements
- Linux environment (WSL/Ubuntu)
